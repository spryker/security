# Http Module
[![Build Status](https://travis-ci.org/spryker/http.svg)](https://travis-ci.org/spryker/http)

Provides HTTP functionality. E.g. `HttpFoundationFormPlugin` or `HttpApplicationPlugin`.

## Installation

```
composer require spryker/http
```
